# portfolio-dimitry

React + R3F portfolio with a modal video for the third frame and GitHub Pages deployment.

## Local Development

- `npm install`: install dependencies
- `npm start`: run dev server at `http://localhost:3000`

## Deploy

- `npm run build`: build production assets
- `npm run deploy`: publish to GitHub Pages (`gh-pages` branch)

Homepage is configured in `package.json` as `https://foukouda.github.io/portfolio-dimitry/`.
